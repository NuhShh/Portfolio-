### 1. E-Commerce Funnel Analysis for Maven Fuzzy Factory

**Overview:**  
Maven Fuzzy Factory, an e-commerce toy store, needed to understand where users were dropping off in their online purchase journey. As the data analyst, I performed funnel analysis using SQL to extract and transform data from multiple tables, then built an interactive Power BI dashboard to visualize user flow and identify conversion bottlenecks.

**Goal:**  
Identify where users drop off in the purchase funnel and provide actionable recommendations to increase conversion rates and revenue.

**Tools/Skills:**  
SQL, Power BI, SQLite, Data Analysis, Data Visualization

**Result**  
Analysis of 140,571 sessions revealed an 8.06% overall conversion rate with the biggest bottleneck at the "Add to Cart" stage (61.1% drop-off). A significant mobile vs desktop conversion gap was identified—desktop converts at 10.05% while mobile only reaches 3.45%. Additionally, two products showed alarmingly high refund rates exceeding 34%. Based on these findings, recommendations include implementing cart abandonment recovery, optimizing mobile UX, and auditing high-refund product quality.
